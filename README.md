# Let's Chill
Let's Chill is a web application that generates random movies from the tv Maze API. More information about the API can be found [here](https://www.tvmaze.com/api)

![screenshot](./images/letsChill.png)


## Built With

- npm node_modules
- Webpack
- tv-Maze API
- HTML, CSS, JavaScript
